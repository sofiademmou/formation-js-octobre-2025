Structure de base d'une page HTML

Cette page démontre la structure d'un document HTML valide.

Explications

  • <!DOCTYPE html> : Déclare le type de document (HTML5)
  • <html lang="fr"> : Elément racine, avec la langue du contenu
  • <html title> : Titre de la page (onglet du navigateur)
